# PowerPlan Switcher

A simple, beautiful Windows app to instantly switch your active power plan right from your system tray or a clean interface.

## Features

* **Fully Portable (Single File)**: Just download and run the `.exe` file! No installer, no setup, and no extra files or folders needed. You can put it anywhere (Desktop, Documents, etc.).
* **Simple Tray Access**: Right-click the icon in your system tray to quickly switch plans without opening the main window.
* **"Ultimate Performance" Auto-Unlock**: If your PC is missing Windows' hidden "Ultimate Performance" plan, the app automatically unlocks it for you.
* **Lightweight**: Built to be fast. Uses almost zero CPU resources and less than 15MB of RAM.
* **Dynamic Taskbar & System Tray Icons**: The app's taskbar and system tray icons change in real-time to match whatever power plan you choose.

---

## How to Use

1. Head over to the **[Releases](https://github.com/nipunyatawara-dev/windows-powerplan-switcher/releases)** tab.
2. Download `PowerPlan Switcher.exe`.
3. Double-click the file to launch it!

*Tip: You can check "Start with Windows" inside the app to let it run quietly in the background in your system tray whenever you log in.*
